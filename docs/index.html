How It Works

How MCP integration works

One command connects depguard to your AI agent. From that moment, every dependency decision is data-driven.

Connect

Add depguard as an MCP server to your AI coding agent. One command, zero configuration, no API keys needed.

claude mcp add depguard

Agent Calls depguard

When your AI agent wants to install a package, it calls depguard's MCP tools automatically — checking 2 advisory databases, 9 attack patterns, and 5 quality dimensions.

depguard_should_use("yaml parser")

Informed Decision

depguard returns a clear recommendation: install, caution, write-from-scratch, or use a native Node.js API. The agent acts on data, not guesswork.

install · caution · native · write
MCP Tools

9 tools your AI agent can call

Each MCP tool returns structured data with token savings tracking. Your agent gets exactly the information it needs to make safe dependency decisions.

◎

depguard_should_use

The primary tool. Agent describes what it needs ("yaml parser") — depguard checks native alternatives first, then scores npm candidates and recommends install, caution, or write-from-scratch.

Start Here
◈

depguard_audit

Full security audit of a specific package. Combines npm + GitHub Advisory Database, analyzes install scripts for 9 attack patterns, suggests fix versions.

Deep Analysis
■

depguard_score

Quick 0–100 score across security, maintenance, popularity, license, and dependencies. When the agent needs a fast quality check without a full audit.

Fast Check
♦

depguard_audit_project

Audit an entire package.json in one call. The agent passes a file path — depguard reads it, detects the project license, and audits every dependency.

Full Project
▲

depguard_search

Search npm by keywords, sorted by depguard quality score. The agent finds the best packages for any intent without web searches.

Discovery
◆

depguard_audit_bulk

Audit multiple packages in parallel. Accepts an array of names or a dependencies object directly from package.json.

Batch Audit
✖

depguard_guard

Pre-install guardian. Verifies the package exists (blocks AI hallucinations), checks for typosquatting via Levenshtein distance, runs a quick audit, and returns allow/warn/block.

Guardian
✔

depguard_verify

AI hallucination guard. Checks if a package name actually exists on npm. Detects typosquatting against 100+ popular packages. Fast enough for every install suggestion.

Anti-Hallucination
✂

depguard_sweep

Dead dependency detection. Scans source files for imports, cross-references with package.json, and finds unused packages with estimated size savings.

Cleanup

Real Threats

These attacks happened

Major npm supply chain incidents and how depguard detects the patterns that made them possible.

Package Year Severity What Happened depguard Detection
event-stream 2018 Critical Malicious dependency stole cryptocurrency wallet keys. 8M weekly downloads compromised. Script analysis flags obfuscated code + credential access patterns in flatmap-stream.
ua-parser-js 2021 Critical Hijacked account published cryptominers and password stealers via postinstall. Install script analysis catches process spawning, external network calls, credential file access.
node-ipc 2022 Critical Maintainer added code to wipe files based on geolocation. Dependency of vue-cli. Advisory database flags GHSA. Script analysis detects file system writes + network conditionals.
colors + faker 2022 High Maintainer sabotaged packages with infinite loops. Broke thousands of projects. Maintenance score drops. Advisory database flags incident. Score falls below threshold.
eslint-scope 2018 Critical Compromised token published version stealing npm tokens from ~/.npmrc. Script analysis detects credential file access (~/.npmrc) and env var exfiltration.
coa + rc 2021 Critical Hijacked accounts. Obfuscated malware in preinstall scripts. Base64 decoding + dynamic code execution detected. Process spawning flagged.
crossenv (typosquat) Ongoing High Names mimicking popular packages. Steal environment variables on install. Low popularity + script analysis flags env var exfiltration. Score far below threshold.
